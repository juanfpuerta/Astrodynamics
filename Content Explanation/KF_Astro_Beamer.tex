\begin{frame}{What is Orbit Determination (OD)?}
    \begin{itemize}
        \item \textbf{Definition:} The process of calculating the exact position and velocity (state vector) of a spacecraft.
        \item \textbf{The Challenge:} Observations (range, Doppler, optical) are inherently noisy and incomplete.
        \item \textbf{Classical Methods:} Batch Least Squares or Differential Correction. Great for ground-based post-processing, but computationally heavy.
    \end{itemize}
\end{frame}

\begin{frame}{Why the Kalman Filter?}
    \begin{block}{Real-Time, Sequential Processing}
        Unlike batch estimators that need chunks of historical data, the Kalman Filter is \textbf{recursive}.
    \end{block}
    \vspace{0.5cm}
    \begin{itemize}
        \item It only needs the \textit{current state estimate}, \textit{current uncertainty}, and the \textit{newest measurement}.
        \item Allows continuous, autonomous updates onboard the spacecraft.
    \end{itemize}
\end{frame}

\begin{frame}{Balancing Physics with Noisy Sensors}
    The Kalman Filter acts as an optimal mathematical referee between two imperfect sources:
    \vspace{0.3cm}
    \begin{enumerate}
        \item \textbf{The Propagated State:} Where the physics models (gravity, drag) predict the spacecraft should be.
        \item \textbf{The Measurement:} Where the noisy sensors say the spacecraft is.
    \end{enumerate}
    \vspace{0.3cm}
    It maintains a \textbf{Covariance Matrix} (uncertainty) to weigh which source to trust at any given second.
\end{frame}

\begin{frame}{Handling "Process Noise"}
    \begin{itemize}
        \item In astrodynamics, there are unpredictable forces (e.g., unpredictable solar flares changing atmospheric drag, or outgassing).
        \item \textbf{Process Noise} tells the filter: \textit{"Assume there are small, unmodeled forces acting on the ship."}
        \item This prevents the algorithm from becoming overconfident in its mathematical model.
    \end{itemize}
\end{frame}

\begin{frame}{The Extended Kalman Filter (EKF)}
    \begin{alertblock}{The Non-Linear Problem}
        The standard Kalman Filter only works for linear systems. Orbital mechanics (inverse-square gravity, exponential drag) are highly non-linear.
    \end{alertblock}
    \begin{itemize}
        \item \textbf{How it works:} It linearizes the orbital dynamics around the current estimate using Jacobian matrices.
        \item \textbf{Industry Standard:} The workhorse of aerospace, used in Apollo and modern GPS receivers.
    \end{itemize}
\end{frame}

\begin{frame}{The Unscented Kalman Filter (UKF)}
    \begin{itemize}
        \item \textbf{When EKF fails:} During atmospheric reentry, highly elliptical orbits, or tumbling spacecraft, linear approximations diverge.
        \item \textbf{How UKF works:} It uses deterministic sample points (\textit{Sigma Points}) propagated through the true, non-linear equations.
        \item \textbf{Advantage:} Highly accurate in extreme dynamic regimes without requiring complex analytical derivatives.
    \end{itemize}
\end{frame}

\begin{frame}{Summary}
    \begin{itemize}
        \item Autonomous spaceflight relies entirely on the Kalman Filter family.
        \item It actively filters sensor noise.
        \item It dynamically adapts to unmodeled physical forces.
        \item Derivations like the EKF and UKF are necessary to handle the non-linear realities of gravity and atmospheric drag.
    \end{itemize}
\end{frame}
